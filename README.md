# XHBD
## 描述
用Python编写的弹窗程序。
## 使用方法
1. 安装Python。
2. 根据自己需要，自行修改bd.pyw。
3. 运行bd.pyw。
## 更多使用方法
### 将脚本打包成exe文件
1. 先执行如下命令安装pyinstaller：
```bat
python -m pip install pyinstaller
```
2. 执行如下命令将脚本打包为exe文件：
```bat
pyinstaller -i xh.ico -n XHBD bd.pyw
```
3. 切换至当前目录下dist\XHBD文件夹，即可执行打包好的XHBD.exe。
## 警告
此程序仅用于娱乐用途，不能将其用于不合规用途。
